# Project Structure - ILM Academy Website

## Directory Overview
The project is a static website built with HTML, Tailwind CSS, and Vanilla JavaScript. It serves as the primary digital presence for ILM Academy in Nairobi.

## Files and Directories

### Root Level
- `index.html`: The landing page featuring hero video, about summary, and program highlights.
- `about.html`: Detailed information about the academy's history, mission, and vision.
- `admissions.html`: Information regarding enrollment, fees, and procedures.
- `programs.html`: High-level overview of all educational programs offered.
- `britishsystem.html`: Dedicated page for the British (UK) Curriculum.
- `american-curriculum.html`: Dedicated page for the American (US) Curriculum.
- `ib-programme.html`: Details about the International Baccalaureate Candidate status.
- `facilities.html`: Visual and descriptive showcase of the academy's facilities.
- `contact.html`: Contact details, business hours, and location information.
- `privacy.html`: Data protection and privacy policy.
- `thank-you.html`: Confirmation page for form submissions.
- `README.md`: Project overview and setup instructions.
- `project_structure.md`: This file, documenting the file organization.
- `TASK.md`: Tracking ongoing and completed development tasks.
- `PLAN.md`: Strategic planning for new features or modifications.

### Assets
- `img/`: Subdirectory for organized image assets.
- `js/`: Subdirectory for JavaScript files.
- `*.jpg / *.mp4`: Various image and video assets used across the site.
  - `ilm logo.jpg`: Primary school logo.
  - `IBVID.mp4`: Educational video.
  - `WhatsApp Video...mp4`: Background hero video.

## Technology Stack
- **Styling**: Tailwind CSS (CDN), Font Awesome 6.4.0.
- **Logic**: Vanilla JavaScript for mobile navigation and scroll effects.
- **Content**: HTML5, Structured Data (JSON-LD).

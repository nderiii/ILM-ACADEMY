# ILM Academy Website

A professional, responsive website for **ILM Academy**, an international school located in Nairobi, Kenya. The academy offers a unique blend of academic excellence and spiritual values, providing U.S., UK-based (British), and International Baccalaureate (IB) curricula.

## 🌟 Overview

ILM Academy is dedicated to nurturing academic achievement alongside spiritual values to develop the leaders of tomorrow. This website serves as the digital front for the academy, providing information about its curriculum, programs, facilities, and admissions process.

### Key Curricula Offered:
- **British Curriculum**: UK-based education system focusing on foundational skills and rigorous standards.
- **American Curriculum**: U.S.-based educational framework emphasizing critical thinking and holistic development.
- **International Baccalaureate (IB)**: Candidate School for the IB Diploma Programme (DP).

## 🚀 Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views using Tailwind CSS.
- **Dynamic UI**: Includes smooth animations (fade-ins, slide-ins) and interactive elements.
- **SEO Optimized**: Meta tags, structured data (JSON-LD) for Google Knowledge Graph, and semantic HTML.
- **Comprehensive Sections**:
  - **About Us**: Mission, vision, and core values.
  - **Programs**: Details for Early Years, Primary, Middle School, and High School.
  - **Admissions**: Information on how to join the academy.
  - **Facilities**: Showcase of the school's state-of-the-art environment.
  - **Contact**: Integrated map and contact form (placeholder) for inquiries.
- **Interactive Elements**: WhatsApp floating button for instant communication and a "Back to Top" feature.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **Tailwind CSS**: Modern utility-first styling.
- **Font Awesome**: Iconography.
- **JavaScript (Vanilla)**: For UI interactions and mobile menu functionality.
- **Google Fonts**: Modern typography (Sans-serif).
